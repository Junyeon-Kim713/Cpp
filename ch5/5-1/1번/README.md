# 인자와 매개변수를 구분하여 설명하시오.
- 인자는 함수를 호출하는 쪽의 지역변수이고 매개변수는 함수 호출 당하는 쪽의 지역변수이다.
# 함수 호출시 자동으로 처리되는 2가지는 무엇인가?
- 매개변수 할당 및 인자값으로 초기화
# 값에 의한 호출로 다른 함수에서 선언된 변수의 값을 변경할 수 없다. 이유를 설명하라.
- 값에 의한 호출은 인자값을 복사해서 매개변수에 대입하는 것이기 때문에 호출한 함수 안에서 다른 함수의 지역 변수의 값을 변경 할 수 없다.
# 주소에 의한 호출을 사용해야하는 경우를 2가지 이상 설명하라.
- 인자로 배열을 보내야 하는 경우, 다른 함수에서 다른 영역의 지역변수의 값을 변경해야 하는 경우, 리턴 값이 2개인 경우
# 인자가 객체인 경우는 어떤 호출방식이 좋은가? 이유도 설명하라
- 주소에 의한 호출이 좋다.
- 객체를 값에 의한 호출할 경우 객체를 복사해서 전달하기 때문에 메모리 손실이 크고 매개변수 객체를 생성할 때 추가적인 함수가 실행되므로 실행 속도도 느리다.
# 함수 반환과정에서 자동으로 처리되는 2가지는 무엇인가?
- 임시 변수 할당 및 리턴 값으로 초기화
# 함수 반환과정에서 임시변수가 필요한 이유를 설명하라
- 반환값은 함수 안에서 선언된 지역변수이기 떄문에 함수가 종료되면 소멸하기 떄문이다.
