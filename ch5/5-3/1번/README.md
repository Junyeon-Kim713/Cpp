# 참조에 의한 호출의 장점을 2가지 측면(메모리사용량, 처리시간)에서 설명하시오  
- 매개변수 할당을 하지 않기 때문에 메모리 사용량이 줄어들고 함수를 호출할 때 추가적인 연산이나 함수 호출이 일어나지 않아 처리시간이 단축된다.
# C++언어 함수의 반환과정을 상세히 설명하라
- 임시 변수 혹은 임시 객체를 만들고 리턴값을 복사해서 임시 변수, 임시 객체에 대입한 뒤 함수는 종료되고 임시 변수, 임시 객체가 함수를 호출한 명령문으로 전달된다.
# 반환형이 참조로 선언된 경우 반환값의 의미를 설명하라
- 임시 변수나 임시 객체를 만들지 않고 함수가 종료될 때 사라지지 않고 존재하고 있는 메모리의 별명, 즉 원본 메모리 공간 전달된다.
# 참조 리턴을 응용하는 예제를 인터넷에서 검색하여 실행해보고 설명하시오
## 큰 값을 가진 변수 직접 수정하는 예제

```cpp
#include <iostream>
using namespace std;

// 두 정수 중 큰 값의 '실체'를 참조로 리턴하는 함수
int& max_ref(int& a, int& b) {
    if (a > b) return a; // a라는 메모리 공간의 별명을 리턴
    else return b;       // b라는 메모리 공간의 별명을 리턴
}

int main() {
    int x = 10, y = 20;

    cout << "수정 전 -> x: " << x << ", y: " << y << endl;

    // 함수 호출 결과(큰 쪽의 실체)에 바로 100을 대입
    max_ref(x, y) = 100; 

    // y가 더 컸으므로, max_ref는 y의 별명을 돌려줬고
    // 결국 y의 값이 직접 100으로 바뀜
    cout << "수정 후 -> x: " << x << ", y: " << y << endl;

    return 0;
}
```
- 참조에 의한 호출과 참조 리턴을 이용해 두 정수를 비교하고 더 큰 값의 값의 참조를 리턴하면 리턴한 참조에 값을 대입하는 예제
