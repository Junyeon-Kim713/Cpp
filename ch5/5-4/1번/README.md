# 일반생성자와 복사생성자의 차이는?
- 일반 생성자는 객체를 생성할 때 자동으로 호출되는 생성자이며 한 객체에 여러 매개변수의 생성자가 선될 수 있다.
- 복사 생성자는 같은 클래스 타입의 원본 객체를 인자로 객체를 생성하거나 객체 생성과 동시에 같은 클래스의 객체로 대입 연산자를 통해 초기화 할 때 호출되며 클래스에 오직 한 개만 선언 가능하다.
# 대입연산과 얕은 복사생성자의 차이는 무엇인가?  
- 대입 연산은 자동으로 정의된 대입 연산자 함수를 통해 얕은 복사를 수행한다.
- 얕은 복사생성자는 대입 연산자 함수처럼 얕은 복사를 하지만 객체 생성과 초기화를 한번에 수행한다.
