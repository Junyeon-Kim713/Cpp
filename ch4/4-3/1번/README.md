# 메모리 누수를 설명하라.
- 동적 할당받은 메모리를 다시 반환하지 않아 힙 메모리 공간이 부족해지는 것이다.
# 메모리 누수가 왜 심각한 문제를 발생시키는지 설명하라.
- 계속해서 메모리 누수가 발생하면 다른 프로그램이 메모리를 할당받지 못한다.
# 메모리 누수를 방지하는 방법을 설명하라.
- 메모리를 할당받은 후 사용이 끝난 메모리 공간은 항상 반환해야한다.
# C++ 스마트 포인터 클래스에 대해 조사하라
- 스마트 포인터 : 메모리 할당과 반환을 자동으로 관리하는 포인터 역할을 하는 객체  
- 종류 : unique_ptr, shared_ptr, weak_ptr
