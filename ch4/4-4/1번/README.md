# string 객체의 문자열 크기 자동 조절 동작 원리  
- string 객체 안에는 size 멤버 변수, capacity 멤버 변수가 존재한다. size는 실제로 사용자가 입력한 문자열의 길이이고 capacity는 string 객체가 힙공간에 빌려놓은 공간의 크기이다.
- 버퍼에서 문자열을 가져올 때 size와 capacity가 같아지면 1.5배 또는 2배 큰 새로운 힙공간을 할당한 후 기존 힙공간에 있던 문자열을 복사해온 후 기존 힙공간은 반환한다.
# 문자열을 입력할 때 마지막에 입력하는 엔터키와 getline 함수의 마지막 인자에 전달하는 구분문자와 차이점  
- 문자열을 입력할 때 마지막에 입력하는 엔터키는 운영체제에 보내는 입력완료 신호이고 버퍼에 있는 문자를 함수가 읽어들이게된다. 또한 버퍼에 \n이 저장된다.
-  getline 함수의 구분문자는 함수가 문자열을 어디까지 읽을 것인지 구분하는 문자이다. 구분문자는 문자열에 포함되지 않는다.
