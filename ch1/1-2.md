# week 1-2
## 1, IDE 란?
- **IDE** : **통합 개발 환경**(Integrated Development Environment), 프로그래머가 프로그램을 만들 때 필요한 모든 도구를 하나로 모아둔 '종합 공구함' 같은 소프트웨어
## 2, IDE의 종류
1. Visual Studio (비주얼 스튜디오)
2. PyCharm (파이참)
3. Visual Studio Code (VS Code)
## 3, IDE의 기능
1. 코드 편집기 (Editor)
2. 빌드 도구 (Build Tool / Compiler)
3. 디버거 (Debugger)
