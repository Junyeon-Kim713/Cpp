# 실습 과제 1  

## 1. 객체의 생성과 소멸의 의미를 설명하라.  
- 객체 생성: 실제 메모리 공간을 할당받아 운영체제가 관리하는 메모리 공간을 프로그램에서 사용할 수 있게됨.
- 객체 소멸: 프로그램에서 사용하던 메모리 공간을 운영체제에 반납하고 더 이상 그 공간을 사용할 수 없음.
## 2. 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라.  
- 지역변수: 함수 내 선언, 함수 안에서만 접근 가능, 함수 호출 시 생성 & 함수 종료시 소멸, 직접 초기화하지 않으면 쓰레기값을 가짐.
- 전역변수: 함부 외부에 선언, 모든 곳에서 접근 가능, 프로그램 시작 후 메인함수 시작 전 생성 & 메인함수 종료 후 소멸, 직접 초기화 하지 않아도 자동으로 0으로 초기화.
## 3. 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사해보라  
- C/C++ 런타임(CRT, C Run-Time) 초기화를 아래 4단계로 진행
- 1. 진입점(Entry Point) 실행
  2. 메모리 초기화 (BSS 영역 0 초기화)
  3. 전역/정적(Static) 객체의 생성자 호출
  4. 실행 인자 준비 및 제어권 이양
## 4. main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라.  
- C/C++ 런타임(CRT)의 종료 및 정리 작업을 역순으로 진행
- 1. atexit() 등록 함수 실행
  2. 전역/정적(Static) 객체의 소멸자 호출
  3. 표준 C 스트림 버퍼 비우기(Flush) 및 닫기
  4. 운영체제(OS)로 제어권 및 종료 코드 반환
