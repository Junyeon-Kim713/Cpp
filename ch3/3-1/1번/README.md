# 구조체와 클래스의 차이점을 설명하라.  
- 구조체는 멤버 변수만 묶을 수 있지만 클래스는 멤버변수는 물론 멤버함수까지 묶을 수 있다.
# 클래스와 객체의 차이점을 설명하라.
- 클래스는 객체를 만들기 위한 사용자 정의 자료형이기 때문에 컴파일할 때만 필요하고 기계어로 변환되지 않는다. 즉 메모리에 실제 할당되지 않는다.
- 객체는 클래스로 만들어낸 변수이고 메모리에 할당된 실체이다. 단순하게 클래스는 자료형, 객체는 그 자료형으로 만든 변수라고 할 수 있다.
# sizeof(클래스명)의 결과값을 출력해보고 결과를 설명하라.
- 클래스의 멤버 변수의 크기의 합이 결과이다. 멤버 함수는 메모리의 효율적인 운영을 위해 **코드 영역**이라는 공용 공간에 단 한번 저장되어 객체들이 같은 함수를 공유하여 사용한다.
따라서, 클래스의 크기에는 반영되지 않는다.
