# C++에서 생성자를 통하여 멤버변수의 초기화를 하는 이유를 설명하라.  
- 객체 내부의 멤버 변수에 대한 직접적인 접근을 막아주고 멤버 함수를 통해서만 접근할 수 있게해, 코드를 안전하게 보호하고 사용자가 이용하기 편리하게 한다.  
# 생성자 작성시 생성자 몸체에서 멤버를 초기화하는 것 보다 멤버 초기화 리스트에서 초기화하는 이유를 설명하라.
- 생성자 몸체에서 멤버를 초기화하면 객체를 만들면서 멤버를 한번 초기화하고 함수를 실행하며 사용자가 원하는 값을 대입해 총 2번의 절차를 거친다. 하지만 멤버 초기화 리스트에서 초기화를 하면
객체가 생성되는 동시에 사용자가 원하는 값으로 바로 초기화되기 때문에 효율적이다.
# 위임, 타겟 생성자로 나누어 작성하는 이유를 설명하라  
- 코드의 중복을 방지하고 유지보수를 용이하게한다.
# 함수 몸체(body)에서 생성자를 직접 호출하면 어떻게 되는지 조사해보라
- 객체 생성 시 추가의 불필요한 임시 객체가 생성되었다가 사라진다.
