# 소스코드 설명  
```cpp
#include <iostream>
```  
- cin, cout, 등 라이브러리 객체의 선언을 포함하고 있는 헤더파일 iostream을 포함   
```cpp
using namespace std;
```  
- std 이름공간에 선언된 모든 이름에 std 생략  
```cpp
int main(void)
```    
- 메인함수 시작  
```cpp
cout << "이름 : 홍길동\n";
```    
- 화면에 이름 : 홍길동\n 를 출력, cout은 화면에 출력해주는 객체, << 삽입연산자 함수  
```cpp
cout << "주소 : 군산시 대학로 558\n";
```    
- 화면에 주소 : 군산시 대학로 558\n 를 출력, cout은 화면에 출력해주는 객체, << 삽입연산자 함수  
```cpp
cout << "학번 : 201012\n";
```    
- 화면에 학번 : 201012\n 를 출력, cout은 화면에 출력해주는 객체, << 삽입연산자 함수  
```cpp
cout << "차종 : 벤츠\n";
```    
- 화면에 차종 : 벤츠\n 를 출력, cout은 화면에 출력해주는 객체, << 삽입연산자 함수  
```cpp
return 0;
```    
- 0을 반환하고 메인함수 정상 종료
