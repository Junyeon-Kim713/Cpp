# 소스코드 설명
```cpp
#include <iostream>
```  
- cin, cout 등의 라이브러리 객체의 선언 포함하고 있 헤더파일 iostream을 포함
```cpp
using namespace std;
```  
- std 이름공간에 선언된 모든 이름에 std::생략
```cpp
int main()
```  
- 메인함수 시작
```cpp
cout << "군산대학교 \"홍길동\" \n";
cout << "\"축하합니다.\" \n";
cout << "100% \\취업률\\ ";
```  
- 화면에 문자열을 출력, cout은 화면에 출력해주는 객체, << 삽입연산자 함수, 이스케이프 시퀀스를 이용한 특수문자 출력
```cpp
return 0;
```  
- 0을 반환하고 메인함수 종
